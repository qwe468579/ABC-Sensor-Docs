# ActiveBiotech - Sensor Documentation

Welcome! This is the documentation for [ActiveBiotech](https://www.activebiotech.com.hk) sensors.

ActiveBiotech develops varies of environment and biometric sensor, all sensor can work with our Android App. For those developer who want to use these sensor on their own application, we also provide an easy to use API. This document describe how to use our Android App and API to communicate with the sensors.

## Sensors

> ### A10 - BLE sensor with IMU, MAG and ALS

> A-Series Sensor is a tiny bluetooth sensor integrated with:

> * Inertial measurement unit (IMU)
> * Magnetometer (MAG)
> * Ambient light sensor (ALS)

> For more specific details please click [here](a-series-sensor.md).

> ### B10 - BLE sensor with IMU, MAG, Air Pressure and Temperature

> B-Series Sensor is a tiny bluetooth sensor integrated with:

> * Inertial measurement unit (IMU)
> * Magnetometer (MAG)
> * Air Pressure Sensor
> * Temperature Sensor

> For more specific details please click [here](b-series-sensor.md).

## Developer Module

> ### A101 - Battery charger with LDO

> A101 is a battery charger module, with the following functions:

> * Battery charging (1-s lithium battery)
> * Voltage Supervision
> * Low drop output (LDO)

> For more specific details please click [here](a101-module.md).

> ### A102 - Battery charger with Buck-Boost

> A102 is a battery charger module, with the following functions:

> * Battery charging (1-s lithium battery)
> * Voltage Supervision
> * Buck-Boost Converter

> For more specific details please click [here](a102-module.md).
